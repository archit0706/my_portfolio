# Archit Kumar Singh - Portfolio

A modern and responsive personal portfolio website built using React and Vite.

## Features

* Modern dark-themed UI
* Responsive design
* Home section with animated developer character
* Education timeline
* Projects showcase
* Skills section
* Contact section with social links
* Smooth scrolling navigation

## Tech Stack

* React.js
* Vite
* CSS3
* React Icons
